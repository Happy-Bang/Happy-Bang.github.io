Excel操作总结

  • 选定单元格区域：Ctrl+Shift+↑/↓ Ctrl+↑/↓：移动到有数据的最后一个单元格

  • 打开单元格形式的快捷键：Ctrl+1

  • 数据类型：文本类型、数值类型、逻辑类型

    文本类型：文本类型就是我们平常所输入的汉字、空格、英文字母。 需要说明的是，阿拉伯数
    字也可以作为文本型数据。例如在Excel2003单元格中输入“12个苹果”，阿拉伯数字12会被当作文本处理

    数值类型： 数值型的数据比如我们平常用的：日期、时间、百分数、会计、科学计数、自定义
    等。

    逻辑类型：Excel中逻辑类型的数据只有两个值，那就是True和False。

注意：我们把1900年1月1日作为日期数据的开始值，用数值1表示

​ 把24个小时用0-1的数值来表示，可以直接通过43882.5表示2020/2/21 12：00：00

  • 在数值类型的约定下时，Excel只为我们存储数值，比如80,100。所以当我们输入0开头的编
    号时，他会默认把0去掉。
  1. 在数值类型的约定下时，输入身份证号，Excel会以为这是个18位的数字，而不是我们传统意
    义上的编号。所以它变成了科学计数法的样子。

0表示通配符，后面加的就是你自定义的格式

  • 快速填充两种方式：

    1、CTRL + E :按规律自动填充

    2、CTRL + ENTER：批量填充同样的内容

  • 快速锁定空单元格：CRTL+G开启定位功能，然后用批量快速填充法进行填充就可以了

  • 拆分合并的单元格：先取消合并，然后定位选中空值，再用公式进行数据的填充

  • 可以选择粘贴的格式，通过右键进行选择性粘贴（转置啊，算数粘贴都有加减乘除，跳过非空数据）

  • 数据验证：可以限制单元格填的数据（序列就是有下拉选项）

  • 条件格式：迅速标记出在条件中的数据，规则可以进行定义和更改

  • 有公式进行复制应该选择性复制值，不然会复制混乱

  • 筛选内容要先打开筛选器

  • 分列功能可以使用符号进行分隔，也可以按固定宽度进行分列

  • 不规则没有分隔符的数据先进行替换，再用分隔符分列

  • 删除重复值功能可以单独删除某一列

  • 一些要做工资条的数据：用辅助列，然后通过辅助列的排序达到原表的排序

  • 快速求和快捷键：ALT +

  • 替换、查找：CTRL + F 替换可以按单元格匹配替换

  • 关系运算符：不等于<> 文本连接符：& 引用运算符：：

  • f4切换单元格引用形式——相对引用：B3 绝对引用：$L$9

  • 逻辑函数and、or

  • if函数：IF 函数是 Excel 中最常用的函数之一，它可以对值和期待值进行逻辑比较。
    因此 IF 语句可能有两个结果。 第一个结果是比较结果为 True，第二个结果是比较结果为 False

    （if函数也可以进行嵌套）

  • 统计函数：

    函数名 意义
    COUNT 返回数字的个数
    COUNTA 返回非空单元格的个数
    COUNTBLANK 返回区域内空格单元数
    COUNTIF j计算区域中满足给定条件的单元格的个数

    SUMIF函数：（条件范围，条件，求和范围）

    EOMONTH(返回一个日期，返回指定月数之前或之后的最后一天)

    • DATEDIF(start_date,end_date,unit)：

      Start_date 为一个日期，它代表时间段内的第一个日期或起始日期。(起始日期必须1900年之后)
      End_date 为一个日期，它代表时间段内的最后一个日期或结束日期。
      Unit 为所需信息的返回类型。

公式：REPLACE(A2,6,5,”*”)：在 abcdefghijk 中，从第六个字符 (f) 开始使用单个字符 * 替换五个字符。

LEFT(范围，截取个数)

RIGHT(范围，截取个数)

  • 查找与引用函数：

    函数名 函数功能
    VLOOKUP 竖向查找
    MATCH 返回查找值的相对位置编号
    INDEX 返回相对位置编号的对应值
    OFFSET 以指定的引用为参照系，通过给定偏移量得到新的引用。返回的引用可以为一个单元格或单元格区域。并可以指定返回的行数或列数。
    INDIRECT 返回由文本字符串指定的引用。

    Reference 必需。 要作为偏移基准的参照。 引用必须引用单元格或相邻单元格区域。否则,
    OFFSET 返回 #VALUE! 。
    Rows 必需。 需要左上角单元格引用的向上或向下行数。 使用 5 作为 rows 参数，可指定引
    用中的左上角单元格为引用下方的 5 行。 Rows 可为正数（这意味着在起始引用的下方）或
    负数（这意味着在起始引用的上方）。
    Cols 必需。 需要结果的左上角单元格引用的从左到右的列数。 使用 5 作为 cols 参数，可指
    定引用中的左上角单元格为引用右方的 5 列。 Cols 可为正数（这意味着在起始引用的右侧）
    或负数（这意味着在起始引用的左侧）。
    高度 可选。 需要返回的引用的行高。 Height 必须为正数。
    宽度 可选。 需要返回的引用的列宽。 Width 必须为正数。

  • INDIRECT：当我们把一个地址带上引号传给函数时，他会应用这个地址里的内容。如果我们只传入地址
    不带上引号，他会用这个地址里储存的地址对应的内容，如果传入的地址里没有地址，则会返回错误
