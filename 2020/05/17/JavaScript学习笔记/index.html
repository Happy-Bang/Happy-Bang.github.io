JavaScript学习笔记（更新ing）

  • JS的组成：

    ECMAScript：ECMAScript 规定了JS的编程语法和基础核心知识，是所有浏览器厂商共同遵守的一套JS语法工业标准。

    DOM：页面文档对象模型，ECMAScript 规定了JS的编程语法和基础核心知识，是所有浏览器厂商共同遵守的一套JS语法工业标准。

    BOM：浏览器对象模型，它提供了独立于内容的、可以与浏览器窗口进行互动的对象结构。通过BOM可以操作浏览器窗口，比如弹出框、控制浏览器跳转、获取分辨率等。

  • JS有3种书写位置，分别为行内、内嵌和外部。

  • 常用的输入输出语句：

    方法 说明
    alert（msg） 浏览器弹出提示框
    console.log(mag) 浏览器控制台打印输出信息
    prompt(info) 浏览器弹出驶入框，用户可以输入

  • 解释型语言（JS）和编译型语言（Java）：

    翻译器翻译的方式有两种：一个是编译，另外一个是解释。两种方式之间的区别在于翻译的时间点不同；

    编译器是在代码执行之前进行编译，生成中间代码文件；

    解释器是在运行时进行及时解释，并立即执行(当编译器以解释方式运行的时候，也称之为解释器)

  • 标识(zhi)符：就是指开发人员为变量、属性、函数、参数取的名字。

  • 变量的数据类型：

    JavaScript 是一种弱类型或者说动态语言。这意味着不用提前声明变量的类型，在程序运行过程中，类型会被自动确定；

    JavaScript 拥有动态类型，同时也意味着相同的变量可用作不同的类型；

  • 数据类型的分类：

    简单数据类型（Number,String,Boolean,Undefined,Null）

    复杂数据类型（object)

  • 在JS中八进制前面加0，十六进制前面加0x

  • 最大值：Number.MAX_VALUE，这个值为：1.7976931348623157e+308

  • 最小值：Number.MIN_VALUE，这个值为：5e-32

  • NaN ：Not a number，代表一个非数值

  • 字符串转义符

    \n：换行符

    \t：tab缩进

    \b：空格

  • 一个声明后没有被赋值的变量会有一个默认值undefined ( 如果进行相连或者相加时，注意结果）

  • 一个声明变量给null 值，里面存的值为空（学习对象时，我们继续研究null)

  • typeof 可用来获取检测变量的数据类型，例如：typeof 18//结果 number

  • 字面量：字面量是在源代码中一个固定值的表示法，通俗来说，就是字面量表示如何表达这个值。

  • 转换为字符串：最常用隐式转换，例如：1+” “

  • 转换为数字型：

    parseInt（string）：转换为整数数值型

    parseFloat（string）：转换为浮点数数值型

    js隐式转换（- * /）：利用算数运算隐式转换为数值型 ‘12’-0

  • 浮点数精度计算问题：var result = 0.1+0.2; //结果不是0.3，而是0.30000000000000004，所以不要直接判断两个浮点数是否相等

  • ++n：前置递增——先自加，后返回值

  • n++：后置递增——先返回原值，后自加

  • ==：赋值 ===：判断两边的值和数据类型是否完全相等

  • 一个例子：（18==‘18’）//true 会自动将字符串类型转换成数字型

  • 短路运算（逻辑中断）：

    原理：当有多个表达式（值）时，左边的表达式可以确定结果，就不再继续运算右边的表达式的值；

    逻辑与：表达式1&&表达式2

    • 如果第一个表达式的值为真，则返回表达式2 （123%456）//返回456
    • 如果第一个表达式的值为假，则返回表达式1 （0&&123） //返回0

    逻辑或：表达式1||表达式2

    • 如果第一个表达式的值为真，则返回表达式1 （123||456） //返回123

    • 如果第一个表达式的值为假，则返回表达式2 （0||456） //返回456

  • 逻辑运算符优先级：先&&后||
